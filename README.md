# BU-EC327-PA1

##C++ code for solving problems relating to data types, operations, if statements, simple loops, and basic functions. Developed as part of EC327: Intro to Software Engineering at Boston University.


###Q1: Shortest Distance from a Point to a Line
A program that takes in three x,y coordinates from console (STDIN). First two coordinates will be used to compute an equation of a straight line passing through each coordinate. The program calculates the shortest distance from the third coordinate to the line and prints the result to console (STOUT).

###Q2: Tip and Tax Calculator
A program that calculates the tip amount, tax amount, and otal costs on a bill. The program allows for four tip levels (10%, 15%, 20%, 25%) and determines if the tip should be calculated on the pre or post-tax amount.

###Q3: Hamming Distance
Program that takes in two numbers between 0-100 (decimals) and computes the Hamming distance between the two numbers represented in base-3 (ternary format). The program will display the Hamming distance in the console.

###Q4: Fire, Water, Grass Game
Program that plays "Fire, Water, Grass." In this game, the user is first prompted to choose "Fire, Water, or Grass." After choosing, the computer randomly generates one of the three options and the program will then display the choice from the computer and determine the winner in console. Fire beats Grass, Grass beats Water, and Water beats Fire.

###Q5: Cryptography
Program that takes in a letter (either lowercase or uppercase) as inputs and prints the next ASCII index of the character with the addition of cyclic wrapping.
